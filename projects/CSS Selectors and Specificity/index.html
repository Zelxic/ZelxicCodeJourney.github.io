CSS Selectors and specificity Reference

CSS Selectors

Selectors Example Example Description
* * Select all elements
.class .hello Select all elements with class="hello"
.class1.class2 .name1.name2 Select all elements with both name1 and name2 within the class attribute
.class1 .class2 .name1 .name2 Select all element with name2 descendent of the elements with name1
#id #myName Select the element with id="myName"
element p Selects all <p> elements
element.class p.hello Selects all <p> elements with class="hello"
element, element div, p Selects all <div> and <p> elements
element element div p Selects all <p> elements inside <div> elements
